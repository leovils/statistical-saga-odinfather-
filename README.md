# The Statistical Saga of Odinfather

A didactic RPG visual novel that teaches Statistics through Norse mythology. The player traverses 7 realms, faces multiple-choice challenges, and accumulates XP — from undergraduate (Apprentice) to graduate level (Master).

## How to Play

1. Visit: **[statistical-saga-odinfather](https://YOUR-USERNAME.github.io/statistical-saga-odinfather/)**
2. Choose a Realm on the map
3. Select difficulty: Apprentice (with hints) or Master (no hints + extra challenges)
4. Answer challenges, earn XP, and unlock viking gear

No installation needed — runs directly in the browser.

## The 7 Realms

| Realm | Topic | Setting | Challenges |
|-------|-------|---------|------------|
| 1 | Quantitative Research | Gates of Midgard | 7 + 2 Master |
| 2 | Descriptive Statistics | Mountains of Jotunheim | 7 + 1 Master |
| 3 | Probability | Temple of the Norns | 7 + 1 Master |
| 4 | Sampling | Land of a Thousand Clans | 7 + 1 Master |
| 5 | Hypothesis Testing | Court of Asgard | 7 + 2 Master |
| 6 | t-Test | Arena of Valhalla | 7 + 2 Master |
| 7 | Equivalence Testing (TOST) | Bridge of Bifrost | 7 + 2 Master |

**Total: 49 challenges (Apprentice) / 60 challenges (Master)**

## Characters

- **Odinfather** — The wise mentor who guides the researcher
- **Thorment** — The antagonist who represents common statistical errors
- **Muninn & Huginn** — Raven helpers that provide hints (Apprentice) and technical details

## Features

- Progressive viking armor reward system — earn gear with each correct answer
- Themed gear unique to each realm (from Scribe's Sandals to the Crown of Bifrost)
- Forge sparkle animation on earning gear
- Tier-based armor appearance by ranking (Captive, Squire, Warrior, Guardian)
- Fully responsive (desktop and mobile)

## Technology

- Pure HTML/CSS/JS — no external dependencies (except Google Fonts)
- Each realm is an independent, self-contained HTML file
- Works offline after first load

## Deploy on GitHub Pages

1. Create a repository on GitHub
2. Upload all `.html` files + `README.md`
3. Go to **Settings -> Pages -> Source: main branch -> Save**
4. Access at `https://your-username.github.io/repo-name/`
